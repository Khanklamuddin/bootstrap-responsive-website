# bootstrap-responsive-website
A modern and responsive website built using *Bootstrap 5*. This project demonstrates the use of Bootstrap components to create a responsive card, grid layout, and services section that works seamlessly across all devices.

## 🚀 Features

- Responsive Navigation Bar
- Responsive Card with Image and Text
- Responsive Grid Layout
- Services Section
- Mobile-Friendly Design
- Bootstrap 5 Components
- Hover Effects and Clean UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 5
